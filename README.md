# 🏋️ GoGetFit

**GoGetFit** is a modern fitness tracking web app that helps users log workouts, track weight progress, and monitor strength gains over time.

---

## 🚀 Features

- 📊 Track body weight over time  
- 🏋️ Log workouts and strength training sessions  
- 📈 Visual progress tracking  
- ⚡ Fast, lightweight web app  
- 📱 Mobile-friendly (can be added to home screen)

---

## 🎯 Purpose

GoGetFit is built to make fitness tracking **simple, fast, and consistent** — no complicated setup, just log your progress and improve every day.

---

## 🛠️ Tech Stack

- HTML5  
- CSS3  
- JavaScript (Vanilla)  
- GitHub Pages (Hosting)
